# 🤖 Gemini Multimodal Chatbot

A smart, modern desktop chatbot application built with **Python**, **CustomTkinter**, and powered by the advanced **Google Gemini API (gemini-2.5-flash)**. This project supports text conversations, image analysis, and voice recording inputs, featuring persistent memory and dynamic theme switching.

---

## ✨ Features
- **Multimodal Inputs:**
  - 💬 **Text Chat:** Seamless real-time text conversation.
  - 📸 **Image Analysis:** Upload any image (`.png`, `.jpg`) and ask Gemini to analyze it.
  - 🎤 **Voice Recording:** Record a 5-second audio clip directly from your mic and send it to the model.
- **Persistent Memory:** Saves your chat history to a local JSON file, allowing you to resume conversations even after closing the app.
- **Dynamic UI/UX:** Modern Desktop Interface with dark/light mode toggle that saves user preference.
- **Robust Error Handling:** Handles API quotas, missing keys, and device recording errors gracefully without crashing.
